# Virtuos Studio — Agence Web Futuriste

Site statique moderne (HTML/CSS/JS) avec GSAP, ScrollTrigger, Lenis et SplitType. Thème sombre premium, navbar avec CTA, section projets en vedette, section tarifs, pages template pour chaque projet.

## Démarrage

- Ouvrez `index.html` dans votre navigateur.
- Les projets détaillés se trouvent dans le dossier `template/` (ex: `template/evasionsignature.html`).

## Technologies et libs

- HTML5, CSS3 (theme sombre, glassmorphism), JavaScript vanilla
- GSAP 3 + ScrollTrigger
- Lenis (smooth scroll)
- SplitType (découpage de texte)

## Structure

- `index.html` — page d'accueil
- `assets/css/style.css` — styles globaux
- `assets/js/main.js` — scripts et animations
- `template/*.html` — pages de présentation des projets (prévisualisation avec iframe)

## Personnalisation

- Modifiez la palette et rayons via `:root` dans `assets/css/style.css`.
- Ajoutez de nouvelles cartes projet dans la section `#projects` de `index.html` et créez une page dans `template/` si besoin.
